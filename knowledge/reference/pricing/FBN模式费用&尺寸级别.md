### 一、 FBN 商品发货尺寸分级表 (Size Classifications)

Noon 系统根据商品加上包装后的外径尺寸（长、宽、高）及发货重量（物理重量与体积重量取大者），将商品划分为以下等级：

|   |   |   |   |   |
|---|---|---|---|---|
|尺寸分层|最长边 (cm)|中边 (cm)|最短边 (cm)|最大计费重量 (kg)|
|小信封 (Small envelope)|≤ 20|≤ 15|≤ 1|0.1|
|标准信封 (Standard envelope)|≤ 33|≤ 23|≤ 2.5|0.5|
|大信封 (Large envelope)|≤ 33|≤ 23|≤ 5|1|
|标准包裹 (Standard Parcel)|≤ 45|≤ 34|≤ 26|12|
|超大包裹 (Oversize)|≤ 130|≤ 34|≤ 26|30|
|特大号 (Extra Oversize)|≤ 130|≤ 130|≤ 130|30.0 (不含玻璃/镜子等易碎品)|
|大件 (Bulky)|超出以上限制|超出限制|超出限制|> 30.0 或任何超出上述尺寸的商品|

_(注：计算体积重与仓储费的单位为立方__英尺_ _CBF，换算公式为：长×宽×高(cm) / 28317。)_

### 二、 FBN 仓储费明细表 (Storage Fees)

FBN 仓储费按月收取，依据商品占用的立方英尺 (CBF) 计算。此外，针对滞销商品有高昂的惩罚性费用：

|   |   |   |   |
|---|---|---|---|
|费用类型|触发条件|阿联酋 (UAE) 费率|沙特 (KSA) 费率|
|月度仓储费|存放于官方仓的所有正常商品|1.5 AED / CBF / 月|2.5 SAR / CBF / 月|
|长期仓储费|存放在仓库内超过 365 天的滞销商品|25 AED / CBF|25 SAR / CBF|
|非销售库存仓储费|质检失败或退回的不可售商品，滞留超 30 天|12 AED / CBF|12 SAR / CBF|
