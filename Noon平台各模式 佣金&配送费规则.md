# Noon平台运营知识库（完整版）

> **数据来源**：Noon官方政策文件（2025年9月1日版 & 2024年Q4新政）
> **适用站点**：沙特阿拉伯 (KSA) / 阿联酋 (UAE) / GCC海湾国家（科威特、卡塔尔、阿曼、巴林）
> **所有费用均不含增值税 (VAT)**：沙特 VAT = 15%，阿联酋 VAT = 5%
> **佣金计费基数**：佣金 = 含税售价 × 佣金比例，平台收取佣金后会额外对佣金收取 VAT

## 一、三大发货模式总览

Noon平台有三种核心发货/履约模式：**FBN（官方仓履约）**、**FBP（海外仓自发货/DirectShip）**、**Global（跨境分销）**。

### 1.1 FBN模式（Fulfilled by Noon / noon Express）

- **定义**：卖家将商品发至Noon官方仓库，Noon负责存储、拣货、打包、尾程配送及售后客服。
- **前端标识**：商品带有 **"noon Express"** 标识，顾客可通过此标签筛选。
- **配送承诺**：通常24小时内发货。
- **核心优势**：转化率平均高出60%；获得参与"每日特价"等促销的独家资格；部分品类享有更低佣金率。
- **成本公式**：`佣金 + FBN出库费 + 月度仓储费 + (潜在长期仓储费)`

### 1.2 FBP模式（Fulfilled by Partner / DirectShip）

- **定义**：卖家将库存备在自己的本地海外仓（或第三方海外仓），订单产生后卖家负责拣货打包，交由Noon物流团队负责尾程派送。
- **核心优势**：库存自主可控，无需承担Noon FBN仓储费，适合多平台铺货。
- **交货方式**：上门揽收 (Pick-up) 或 卖家自送至Noon网点 (Drop-off)，两种方式费率不同。
- **成本公式**：`佣金 + FBP尾程配送费 - 运费补贴 + (各类隐性/惩罚费用)`

### 1.3 Global模式（跨境分销 / 非居民卖家）

- **定义**：依托Noon Global项目，以阿联酋(UAE)的FBN或FBP仓储为起运点，自动将商品跨境分销至沙特(KSA)及其他海湾国家。
- **适用对象**：阿联酋境内卖家，以及在阿联酋注册的非居民卖家(Non-Resident Sellers，如中国出海企业)。
- **模式本质**：不是独立的第三种履约方式，而是在FBN/FBP基础上叠加的"跨境分销层"。
- **结算货币**：最终结算款项以 AED（阿联酋迪拉姆）支付。

### 1.4 三种模式核心差异速查表

| 对比项 | FBN模式 | FBP模式 | Global模式 |
|--------|---------|---------|-----------|
| 库存管理 | Noon官方仓 | 卖家自有仓 | 基于UAE的FBN/FBP仓 |
| 核心物流费 | 出库费（按件，基于尺寸/重量/ASP） | 直发运费（按订单，基于重量） | 本地费用 + 跨境关税/清关费 |
| 月度仓储费 | 有 | 无（仅不可售库存） | 跟随底层模式（FBN有/FBP无） |
| 运费补贴 | 不适用 | 有（按订单金额阶梯） | **无（零补贴）** |
| 配送标识 | noon Express | 无特殊标识 | Global标签 |
| 覆盖市场 | 本国站点 | 本国站点 | UAE → KSA + 科威特/卡塔尔/阿曼/巴林 |
| 佣金优惠 | 部分品类更低 | 标准费率 | 跟随底层模式 |
| 退货成本 | 出库费不退还 | 配送费不退还 | **灾难级**（关税+VAT+双程运费均不可收回） |

## 二、平台佣金（Referral Fees）详细对照表

### 2.1 佣金计费规则类型说明

Noon佣金存在两种不同的阶梯计算逻辑，理解差异至关重要：

- **超额累进制**：类似个税，仅对超出阈值的部分适用更低费率。例如手表售价6,000 SAR，前5,000按15%收，超出的1,000按5%收。
- **门槛跳变制**：按商品总价决定整体适用哪档费率。例如手机售价800 SAR(>750阈值)，全额按5.5%收（而非前750按6%后50按5.5%）。

## 三、FBN尺寸分层标准与计费重量

### 3.1 尺寸分层定义

FBN模式在计算发货计费重量时，会固定加上标准"包装重量"。

> **计费发货重量** = MAX(物理净重, 体积重) + 包装附加重量
> **体积重(kg)** = 长(cm) × 宽(cm) × 高(cm) / 5000

| 尺寸分层 | 最大外径尺寸 (cm) | 包装重量附加值 | 最大计费发货重量 |
|---------|-----------------|-------------|-------------|
| 小信封 (Small Envelope) | ≤20 × ≤15 × ≤1 | +20g | 0.1 kg |
| 标准信封 (Standard Envelope) | ≤33 × ≤23 × ≤2.5 | +40g | 0.5 kg |
| 大信封 (Large Envelope) | ≤33 × ≤23 × ≤5 | +40g | 1.0 kg |
| 标准包裹 (Standard Parcel) | ≤45 × ≤34 × ≤26 | +100g | 12.0 kg |
| 超大包裹 (Oversize) | ≤130 × ≤34 × ≤26 | +240g | 30.0 kg |
| 特超大件 (Extra Oversize) | ≤130 × ≤130 × ≤130 | +240g | 30.0 kg |
| 大件/笨重件 (Bulky) | 超过上述任一限制 | +400g | >30.0 kg |

### 3.2 仓储分类体系

**按商品类型**：高价值商品(HRV：手机/平板/笔记本)、服装与鞋履、温控商品(食品杂货类)。
**按商品尺寸**：标准件(最长边≤45cm，重量≤12kg)、超大件、大件(长>130cm或重>30kg)。

> 不同类型和尺寸的仓储空间独立配额，不能相互调剂。

## 五、FBP尾程配送费（DirectShip Shipping Fee）

> **计费重量判定**：严格取物理实重(kg)与体积重(长×宽×高cm / 5000)二者中的最大值作为计费重量。

## 十、FBN运营关键政策（争议与赔偿）

### 10.1 体积计量争议政策（VM Dispute）

如果卖家发现被Noon收取的出库费异常（通常因入库时3D扫描仪测大了体积）：

| 规则 | 详情 |
|------|------|
| 误差容忍度 | 测量值与实际值偏差在 **5%以内** 的争议将被Noon **直接驳回** |
| 时效限制 | 对于同一个P-Barcode，卖家每 **60天** 只能发起一次重新测量的争议 |
| 结果追溯 | 争议成功后，系统仅退还**过去30天内**多收的运费，并对未来订单生效 |

### 10.2 丢失与损坏赔偿（Lost & Damaged Reimbursement）

**仓库丢失**：系统每月自动对账，但有 **3个月的缓冲期**。例如第1个月丢失，若第5个月仍未找到，才会打款赔偿。

**客户退货损坏定级制度**：对于非卖家原因导致的退货损坏，Noon仓库接收后进行定级(Grading)：

| 等级 | 损坏程度 | 赔偿比例 |
|------|---------|---------|
| Grade A | 完好如新 | 0%（不赔偿，商品重新上架） |
| Grade E | 中度损坏 | 25% |
| Grade X | 全损 | 100%（全额赔偿） |

> 卖家无法拿回原物，Noon会根据定级直接给予折价赔偿。

### 12.1 订单取消率（Cancellation Rate）

| 指标 | 标准 |
|------|------|
| 红线 | 必须低于 **2.5%** |
| 考核周期 | 7天 |
| 算法 | (卖家主动取消 + 移交前标记为丢失的订单) / 总订单数 |

### 12.2 发货延迟率（Late Shipment Rate）

卖家必须在设置的Processing Time（如Same Day, 1_day）内完成系统状态更新和实物交接。严重延迟不仅影响转化率，还可能触发取消罚款。

### 13.1 退货窗口与流程

## 十四、Global跨境分销模式专项规则

### 14.1 跨境销售资格与物理红线

不是所有商品都可参与跨境销售，必须满足以下严格条件，否则将被系统自动拦截并保持离线(Offline)状态：

| 限制类型 | 具体要求 |
|---------|---------|
| 履约模式限制 | 只有 **FBN(官方仓)** 和 **FBP(DirectShip Express快速自发货)** 模式下的商品有资格 |
| 重量红线 | 实际重量或体积重 **>40 Kg** 禁止跨境 |
| 尺寸红线 | 最长边 **>120 cm** 禁止跨境 |
| 品类红线 | 被Noon定义为**大件(Bulky)** 的商品禁止跨境 |
| 合规限制 | 在目的国海关被列为限制或违禁品的商品不可跨境 |

### 14.2 自动定价公式（UAE → KSA）

卖家无需手动设置目的国售价，系统基于阿联酋基础报价(UAE Base Price)自动计算：

1. **第一步（剥离阿联酋税）**：UAE售价 ÷ 1.05（剔除5% VAT）
2. **第二步（转为美元中转）**：按实时汇率转为USD（例如÷3.67），得出税前全球价(pre-VAT Global Price)
3. **第三步（目的国加成）**：转换为SAR，系统自动加上沙特进口关税及清关杂费
4. **第四步（加征沙特税）**：加上沙特当地15%的VAT，形成展示给沙特买家的最终价格

### 14.3 费用代扣（Import VAT Recovery）

Noon负责代收代缴目的国的所有关税和增值税。在卖家财务账单中，以"Import VAT recovery(进口增值税代扣)"名义从结算款中扣除。卖家无需自己去目的国交税。

### 14.4 跨境运费与补贴政策

| 项目 | 详情 |
|------|------|
| 跨境FBP运费补贴 | **零(Zero Reimbursement)**。卖家不会获得任何运费补贴 |
| 原因 | Noon承担了昂贵的国际段干线与清关物流成本 |
| 其他GCC国家 | 发往科威特/卡塔尔/阿曼/巴林，在标准佣金外额外收取相关税费 |

### 14.5 售后与质保政策差异（跨境特有）

**质保责任的两极分化**：

| 目的国 | 质保责任 | 换货支持 |
|--------|---------|---------|
| 沙特 (KSA) | **必须**遵守沙特当地保修法律，提供合规质保 | 支持 |
| 其他GCC国家 | **平台免责红利**：不提供任何质保支持，买家无权申请质保 | **不支持换货** |

**跨境退货的沉没成本（对非居民卖家）**：

一旦发生退货，以下费用**绝对不可收回**：
- 已缴纳的进口增值税 (Import VAT)
- 海关关税 (Customs)
- 跨境双程运输费用 (Shipping charges)

> 跨境退货周期(TAT)会因运输距离和海关流程被大幅拉长。

### 14.6 强制性上架合规要求

若要成功激活Global标签并让商品在沙特等地展示，阿联酋店铺的SKU必须补充以下信息（缺一不可）：

| 要求项 | 详情 |
|--------|------|
| 沙特12位海关编码(HS Code) | 必须在ZATCA（沙特天课、税务和海关总署）关税表中查找并填入。**因HS Code错误导致的海关扣件或罚款，卖家承担100%责任，Noon概不负责** |
| 原产国(COO) | 必须明确标注制造或设计国（每个SKU只能填写一个COO） |
| 税务开票要求（非居民卖家） | 上传价格必须含税(VAT)；出口时需开具"零税率纳税发票(Zero-rated Tax Invoice)"，Noon将发票存放在买家账户中 |

## 十六、模式选择决策矩阵

### 16.1 成本核算公式

**FBN总成本** = 佣金 + FBN出库费 + 月度仓储费 + (潜在长期仓储费)

**FBP总成本** = 佣金 + FBP尾程配送费 - 运费补贴 + (订单取消费风险) + (不可售库存仓储费风险)

**Global总成本** = 底层模式成本(FBN或FBP) + 进口关税 + 进口VAT + 跨境运输费 - 零运费补贴

### 16.2 模式选择建议

| 场景 | 推荐模式 | 原因 |
|------|---------|------|
| 小件轻货、高频动销 | FBN | 出库费低于直发运费；noon Express流量加权；部分品类佣金更低 |
| 大件重货 | FBP | 避免高额出库费和仓储费 |
| 长尾慢销品 | FBP | 无月度仓储费压力 |
| 多平台铺货 | FBP | 库存自主可控 |
| 面向沙特市场的阿联酋卖家 | Global(FBN底层) | 自动定价和清关；FBN底层享受noon Express标识 |
| 退货率高的品类跨境 | 谨慎使用Global | 跨境退货成本为沉没成本，绝对不可收回 |
| 出口至科威特/卡塔尔等 | Global | 无质保义务，平台免责红利 |

### 16.3 关键风控提醒

1. **沙特站15%的VAT**是巨大成本因素，定价必须计入
2. **FBP体积重陷阱**：体积重=长×宽×高/5000，轻泡货可能被按体积重收费
3. **FBN包装附加重量**：系统自动加20g~400g包装重，影响出库费档位
